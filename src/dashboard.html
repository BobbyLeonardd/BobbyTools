Panduan Cara Pakai

1. 9Router Mode + Web Dashboard (Auto-Rotate & Load Balancing)

Mode ini bikin BobbyTools jadi Universal Gateway. Cocok kalo lu punya banyak API Key gratisan (ternak akun) dan males gonta-ganti manual pas kena limit (429).

Langkah-langkah untuk CLI lo:

Mac/Linux/GitBash: export OPENAI_BASE_URL="http://127.0.0.1:13337/v1"
Windows Powershell: $env:OPENAI_BASE_URL="http://127.0.0.1:13337/v1"
Windows CMD       : set OPENAI_BASE_URL="http://127.0.0.1:13337/v1"

Panggil CLI lu pake format provider/model (misal: opencode -m groq/llama3-70b-8192).

✨ Magic v3.0.0 (God-Tier Features):

  • Smart Load Balancing: Kalau akun A limit (429), router otomatis muter nyoba pake akun B saat itu juga. CLI lu dapet response sukses tanpa pernah sadar ada error!
  • Universal Fallback: Kalau SEMUA akun Groq lu limit, router bakal nyari provider lain (misal OpenRouter) yang punya model yang sama, lalu langsung diproses otomatis tanpa putus!

2. Classic Launcher (Buat Single Account)

Kalo lu cuma mau jalanin 1 akun dan males ribet set environment variables, tinggal pake perintah bobby di terminal dan pilih 🚀 Start Session. BobbyTools yang bakal nyuntik variabelnya ke memori secara gaib buat lu, terus langsung ngebuka CLI lu.

Pro tip: Ketik bobby go di terminal buat langsung ngegas sesi terakhir tanpa lewat menu!
