Panduan Lengkap (Biar Lo Ga Keder)

1. Mode Web (Router) - Mode Setengah Dewa 🌟

Mode ini alasan utama kenapa BobbyTools diciptain. Kalo lo capek dikit-dikit kena limit (429) gara-gara pake API gratisan, mode ini obatnya. BobbyTools bakal jalan di *background* komputer lo sebagai "Middleman" atau router.

Cara Setup:

  • Tambahin Provider di menu sebelah. (Misal: Groq, Anthropic, Gemini).
  • Di dalem Provider itu, tambahin Akun sebanyak yang lo punya. Masukin semua API Key lo dari akun A sampe Z.

Cara Nyambungin ke CLI Ngoding (Aider / Opencode / Cursor):

Lo cuma perlu ngibulin CLI lo biar dia ngira BobbyTools itu adalah server aslinya. Buka terminal lo, copas ini:

Mac/Linux: export OPENAI_BASE_URL="http://127.0.0.1:13337/v1"
Windows PS : $env:OPENAI_BASE_URL="http://127.0.0.1:13337/v1"

(Note: Gak usah repot-repot ngeset OPENAI_API_KEY, mau diisi asdf doang juga bodo amat, karena yang nyuntikin API Key asli itu si BobbyTools!)

Cara Ngejalaninnya:

Panggil CLI lo pake format provider/model biar router tau lo mau nembak ke mana.

opencode -m groq/llama3-70b-8192

✨ Keajaibannya (The Magic):

  • Auto-Rotate & Smart Load Balancing: Kalau Akun 1 lo kena limit 429, Bobby diem-diem bakal nge-retry pake Akun 2 tanpa ngasih tau terminal lo. Terminal lo taunya cuma sukses mulus, padahal di belakang layar Bobby lagi kerja keras.
  • Universal Fallback: Kalau sial banget SEMUA akun Groq lo abis, Bobby bakal nyari otomatis di provider lain yang punya model llama3-70b (kalo lo sempet masukin). Canggih kan?

2. Mode Klasik (CLI Launcher) - Mode Magasan 💤

Kalo lo cuma pake 1 akun doang dan males buka web atau ribet masukin environment variable, tutup web ini, buka terminal, ketik bobby.

  • Pilih 🚀 Start Session.
  • Tinggal klik-klik: Pilih Provider → Pilih Akun → Pilih Model.
  • BobbyTools bakal otomatis ngejalanin target CLI lo (misal opencode) dan API Key lo otomatis kesuntik di memori. Lo gak perlu ngapalin sintaks export macem-macem.

Pro tip: Besok-besoknya lo tinggal ketik bobby go di terminal. Bobby langsung ngebuka sesi terakhir lo. Bener-bener buat pemalas.

📚 Daftar Perintah (Commands) BobbyTools

  • bobby
    Ngebuka menu utama interaktif (buat nambah akun, pilih model, dll).
  • bobby go
    Jalan pintas orang malas. Langsung ngebuka sesi terakhir lu tanpa lewat menu klik-klik lagi.
  • bobby serve
    Nyalain Web Dashboard di depan layar (foreground). Kalo terminalnya lu tutup, servernya ikut mati.
  • bobby serve-bg
    Nyalain Web Dashboard di belakang layar (background/daemon). Terminal bebas lu tutup, server tetep idup dan otomatis ngebukain browser.
  • bobby list
    Nampilin daftar lengkap semua Provider dan API Key (Akun) yang udah lu simpen, tanpa masuk ke menu.
  • bobby update
    Ngasih tau instruksi cara update BobbyTools ke versi paling baru dari NPM.
  • bobby -v
    Ngecek versi BobbyTools yang lagi lu pake sekarang.
  • bobby -h
    Nampilin contekan/bantuan.
